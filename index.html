You're a couple steps away from launching SATURN for the first time.

  1. Transfer the saturn directory to your server.
  2. Create a MySQL database named saturn.
  3. Open saturn/system/socket.txt and enter your MySQL socket/address without extra lines or whitespace.
  4. Go to http://www.yoursite.com/saturn/startup.php.
  5. Enter the access key saturna3setupak.jc411, your MySQL socket/address, and your saturn database username and password.
  6. If the final progress line looks like...
    • Database format successful. All tables created., your database has been formatted.
    • Unable to..., an error occurred. Check all MySQL credentials.
  7. Once your database has been formatted, give yourself a pat on the back; you have installed Saturn CMS.
